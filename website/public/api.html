Hello this is very very very good api , it's just for getting releases,versions and download links :)

End Points :

GET /versions : returns the last 30 releases from github

GET /versions/latest : get the latest release

GET /versions/latest/bin/windows : get the latest application binary for windows (not the installer)

GET /versions/latest/bin/linux : get the latest application binary for linux (not the installer)

GET /versions/compare : compare your version with the latest version
Search params : version (the version you want to compare)
